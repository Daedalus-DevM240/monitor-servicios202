Key Features:

TCP Connectivity Checks: Attempts to establish socket connections to target services.

Boolean Status Reporting: Returns True if the service is reachable and False if it is closed or inaccessible.

Modular Design: Easily integrable into larger automation workflows or monitoring dashboards.

Technologies Used:

Python: Core logic and networking.

WSL2/Linux Environment: Development and deployment environment.

Git/GitHub: Version control and code repository.
